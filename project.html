Smart home security camera using Python and OPENCV (FINAL YEAR PROJECT)

A security camera used to detect an intruder or an unknown person in absence of people staying in their home.
Hardware implementations include Arduino Nano, Servo motor, connecting wires, LEDs and webcam.
Software used are Arduino IDE and python IDLE.
OPENCV python library is used to solve computer vision problems.
Haar cascade algorithm is implemented to identify different types of the human faces.
